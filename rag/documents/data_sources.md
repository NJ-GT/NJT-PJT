# 데이터 출처와 사용 파일

대시보드는 `0430` 폴더의 최종 분석 산출물을 중심으로 구성된다.

주요 파일은 다음과 같다.

- `0430/최종테이블0429.csv`: 시설별 최종 위험 점수, 변수, 군집 라벨, 좌표를 포함한다.
- `0430/시설별_화재위험_성적표_0429_v3.csv`: 시설별 화재위험 성적표 형태의 보조 테이블이다.
- `0430/가중치군집_대표모형_최종결과표.csv`: 군집별 대표 공간모형 결과를 담는다.
- `0430/가중치군집_공간모형_비교표.csv`: 군집별 공간모형 비교 결과를 담는다.
- `0430/공간 모델 선택.png`: 공간모형 선택 결과 이미지이다.
- `0430/군집별 피처 평균값-중앙값 비교.png`: 군집별 주요 변수 평균과 중앙값 비교 이미지이다.

대시보드에서 참조하는 추가 데이터는 다음과 같다.

- `원본데이터/서울시 외국인관광도시민박업 인허가 정보.csv`
- `원본데이터/서울시 관광숙박업 인허가 정보.csv`
- `원본데이터/서울시 숙박업 인허가 정보.csv`
- `data/서울10구_숙소_소방거리_유클리드.csv`
- `data/소방서_안전센터_구조대_위치정보_2025_wgs84.csv`
- `road_width_10gu/data/seoul_road_width_10gu_official_only_roads.csv`
- `road_width_10gu/data/seoul_road_width_10gu_official_complete_lines.geojson`
- `road_width_10gu/data/seoul_road_width_10gu_osm_lines.geojson`
- `road_width_10gu/data/seoul_road_width_10gu_osm_line_unmatched.csv`

답변에서는 0430 최종 산출물 기반이라는 점을 우선 설명하고, 원자료나 도로폭 데이터는 보조 참조 자료로 구분해서 설명한다.
