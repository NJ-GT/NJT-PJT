# 최종 상가+숙박 개수밀도 구현자료

최종 HTML은 프로젝트 안에 하나만 남겨두기 위해 이 폴더에 복사하지 않았습니다.

최종 HTML 위치:

`data/건물상가밀집도_10개구_0417/서울10개구_건물상가_50,000sqm_밀도지도.html`

## 폴더 구성

- `01_입력테이블/상가숙소_격자자료`
  - 기존 상가+숙박 격자별 밀집도 CSV/GeoJSON
  - 기존 컬럼정의 및 산출요약
- `01_입력테이블/건물원본_shp`
  - 최종 집계와 실제건물 표시 생성에 사용한 필터링 건물 SHP 세트
- `01_입력테이블/보정테이블`
  - 밀도 0 제외 건물 목록 CSV
- `02_최종산출테이블`
  - 최종 HTML에 들어간 격자별 시각화 CSV
  - 최종 격자별 밀도 GeoJSON
  - 최종 컬럼정의 및 산출요약
- `03_HTML보조파일`
  - 실제건물 점 표시용 JS
- `04_재생성스크립트`
  - 최종 HTML과 산출 테이블을 다시 생성한 Python 스크립트

## 최종 표시 지표

최종 지도 색상 기준은 `상가+숙박 개수/ha` 하나입니다.

계산식:

`상가+숙박 개수/ha = (상가수 + 숙박시설수) / 5ha`

호버와 팝업에는 면적과 층수 기반 참고값도 함께 표시됩니다.
